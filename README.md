# 📊 Customer Revenue & Churn Overview Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard for analyzing customer behavior, revenue performance, and churn impact in a telecom business environment.

The dashboard is built for a fictional company **NexLink Telecom** to simulate real-world business scenarios and provide actionable insights.

---

## 🎯 Objectives
- Analyze customer distribution across different segments
- Identify key revenue drivers
- Measure revenue loss due to churn
- Support data-driven decision making

---

## 🛠️ Tools & Technologies
- **Power BI** → Data visualization & dashboard design  
- **Excel (Power Query)** → Data cleaning & transformation  
- **CSV File** → Data source  

---

## 🔄 Data Preparation (Power Query)
Performed full data cleaning and transformation using Power Query:

- Handled missing values (categorical & numerical columns)
- Standardized and cleaned categorical data
- Split and formatted Customer ID
- Created calculated columns (e.g., Age Groups)
- Ensured data consistency and usability

### ✅ Dynamic Data Handling
The dashboard is directly connected to the CSV file.  
Any updates to the dataset are automatically reflected in Power BI after refresh.

---

## 📊 Key Metrics (KPIs)
- Total Customers  
- Total Revenue  
- Lost Revenue  

---

## 📈 Dashboard Features
- Customer segmentation by **Contract Type**
- Revenue analysis by **Contract**
- Revenue distribution by **Internet Type**
- Interactive filters (Internet Type, Offer, Age Group)
- Clean UI with consistent color theme

---

## 💡 Key Insights
- Customers on Month-to-Month contracts represent the largest segment  
- Long-term contracts generate more stable revenue  
- Fiber Optic customers contribute the highest share of revenue  
- Customer churn has a noticeable impact on revenue loss

This dashboard helps stakeholders to:

- Monitor revenue performance
